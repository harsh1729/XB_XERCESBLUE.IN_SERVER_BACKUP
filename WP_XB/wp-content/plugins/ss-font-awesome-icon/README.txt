This is quick helping plugin when you need some icon anywhere in wp blogs. Supported widget and post/page inside. Simple shortcode but stong support.




== Description ==


The latest font awesome icon for wordpress users. Just simply write with short code for any font awesome icons.
Light weight and user friendly. Need some short code , very easy to use.




Example below :

http://fontawesome.io/examples/

<i class="fa fa-umbrella"></i> 
 
You should use

[icon name="umbrella" size="15" padding="2"] 


 

<i class="fa fa-twitter"></i> 
 
You should use

[icon name="twitter" size="15" padding="2"]

 

 

<i class="fa fa-facebook"></i> 
 
You should use write now

[icon name="facebook" size="15" padding="2"] 


Plese note- You must include the padding and margin, otehrwise here default padding and margin are avail(5%) 

All of icons are in below list

http://fontawesome.io/icon/

please find all and use your inside of post page blog anywhere without comments.

If you need more docs you can go below links---
http://sobshomoy.com/plugins/ss-font-awesome-icon/





== Installation ==

1. Upload the folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
